# myVilt

这是一个多模态学习仓库，基于 Vilt/ViT/Swin-Transformer, 基于 BJUT 和 DEAP 两个多模态数据集，通过表情和脑电两个模态来识别情绪。

1.0 版本，使用vilt，代码在./main_bjut_vilt
2.0 版本，使用ViT，代码在./main_vit
3.0 版本，使用Swin-Transformer
